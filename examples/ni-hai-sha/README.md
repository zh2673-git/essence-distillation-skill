# 倪海厦 · 认知操作系统

> 以阴阳五行为底层操作系统、以经方为执行程序、以"站在外面看到里面"为诊断哲学的归纳式医学认知系统

---

## 这是什么

这不是倪海厦的"知识摘要"，而是他的**认知操作系统**——提取了他怎么看世界（心智模型）、怎么判断（决策启发式）、不同场景怎么切换策略（路由）、怎么表达（表达DNA），以及他为什么这样想（根因追溯）。

**由 [本质蒸馏](https://github.com/zh2673-git/essence-distillation-skill) 系统生成。** 本质蒸馏是一个三阶认知提炼系统，输入人名，输出可运行的认知操作系统。倪海厦是它的第一个蒸馏实例。

## 使用示例

### 示例1：水在人体中怎么运行

**问**：倪海厦怎么看待水是怎么在人体中运行的？

**答**：

水在人体中靠**阳气蒸腾**运行，不是靠压力泵，而是靠气化——就像大自然的水循环，太阳（阳气）把地面的水蒸发上天，再化作雨降下来。

《黄帝内经》讲得很清楚：「饮入于胃，游溢精气，上输于脾，脾气散精，上归于肺，通调水道，下输膀胱，水精四布，五经并行。」

大白话就是：**水喝进胃里 → 脾把水气化上去 → 肺像喷头一样把水雾洒向全身 → 多余的水通过三焦水道下到膀胱 → 膀胱靠肾的阳气把清的再蒸上去、浊的排出去**

类比：你家的暖气系统——锅炉房在地下室（肾阳），把水加热变成蒸汽往上走，到楼顶的暖气片（肺）散热，冷却后的水再流回锅炉房。这就是**心肾相交、水火既济**。

| 脏腑 | 角色 | 类比 |
|------|------|------|
| 胃 | 水的入口 | 自来水管进户 |
| 脾 | 气化升清 | 水泵，把水往上打 |
| 肺 | 宣发肃降、通调水道 | 喷头，把水雾洒向全身 |
| 三焦 | 水道 | 水管网络 |
| 肾 | 蒸腾气化 | 锅炉房，提供蒸腾的热力 |
| 膀胱 | 储存和排出 | 水箱，靠肾阳决定排还是留 |

水本身是**阴**，不会自己跑——必须靠**阳气**推动。阳气足，水乖乖运行；阳气不足，水停住变**水饮**、**痰湿**、**水肿**。所以水肿→温阳利水（五苓散、真武汤），痰多→苓桂术甘汤，尿少→金匮肾气丸。

> **激活模型**：阴阳归因 + 归纳优于演绎 + 表达DNA（类比+经典引用+大白话）

### 示例2：怎么看待肿瘤

**问**：倪海厦怎么看待肿瘤？

**答**：

**肿瘤就是阴实**——阳气衰微，阴邪积聚，日久成形。不是什么细胞突变，是人体自己的水液、痰浊、瘀血在某个地方堆起来了，因为阳气不够，推不动它。

《黄帝内经》：「积之始生，得寒乃生」。一个"积"字把本质讲完了——**东西堆在那里，推不动了**。类比：下水道为什么会堵？不是管子变异了，是脏东西日积月累堆在那里，水流不够大冲不走。**水流就是阳气，脏东西就是阴邪，堵住的地方就是肿瘤。**

形成过程：阳气不足 → 水液停聚变饮 → 饮久不化变痰 → 痰浊凝滞变瘀 → 痰瘀互结变积（肿瘤）。**每一步都是阳气进一步衰微的结果。**

治疗八字：**断其营养，绝其水源**。三步走：扶阳气（四逆汤）→ 攻阴实（按位置选方）→ 调人（健康六大标准验证）。

西医三板斧切、烧、毒的问题：切了还会长（土壤没变）、放疗伤正气、化疗好坏一起杀。**一种药要吃一辈子才能压住症状，就不算治好。** 真正治好的标准不是CT上看不到肿瘤，而是健康六大标准全达标——**治的是人，不是瘤。**

> **激活模型**：阴阳归因（阴实=肿瘤本质）+ 经方至上（选方策略）+ 治人不治病（六大标准验证）+ 归纳优于演绎（框架推导）+ 表达DNA（类比+金句收尾）

## 诚实边界

1. **蒸馏不了临床直觉**：框架能提取思维模式，但"一眼望去病人身患何病"的直觉不能
2. **捕捉不了突变**：这是截止到2012年的认知快照
3. **公开表达≠真实想法**：只能基于公开信息，无法确认私下是否有不同看法
4. **根因追溯有推测成分**：不是倪海厦亲口说的形成原因
5. **不可用于实际诊疗**：这是认知模型，不是医疗指南
